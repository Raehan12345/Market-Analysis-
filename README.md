# Market Data Analysis

A quantitative research project focused on processing high-frequency market data and analyzing market microstructure.

## Objective
Download historical market data from free sources, clean and process tick data, calculate volatility surfaces, analyze order book dynamics, and discover patterns in market microstructure. All findings and visualizations are documented in Jupyter notebooks.


## Project Structure
* `data/`: Local storage for raw and processed tick data.
* `notebooks/`: Jupyter notebooks containing exploratory data analysis and documented findings.
* `src/`: Core Python modules for data fetching, cleaning, and mathematical modeling.

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment.
3. Run `pip install -r requirements.txt`.
4. Launch `jupyter lab` to view the notebooks.
